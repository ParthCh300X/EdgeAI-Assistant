# Edge AI Assistant

A minimal, on-device AI assistant designed for fast, privacy-focused task execution without heavy cloud dependency.

## Overview
Edge AI Assistant is an Android application that integrates essential daily utilities into a single, lightweight interface. It emphasizes offline capability, low latency, and a clean user experience.

## Features
- Voice input and command handling  
- Expression calculator  
- Unit converter  
- Text summarization  
- Image text extraction (OCR)  
- Local app launcher  
- Notes and reminders  
- Alarm and scheduling utilities  
- Search history tracking  

## Tech Stack
- **Frontend:** Jetpack Compose (Kotlin)  
- **Backend Logic:** Kotlin  
- **AI/ML:** On-device processing (Edge AI principles)  
- **Libraries:** Android SDK, ML/OCR integrations  

## Architecture
- Modular intent-based system  
- Scalable feature separation  
- Optimized for low resource consumption
